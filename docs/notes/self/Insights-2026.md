## 沟通

- 不要做伤害孩子安全感的事情，教育孩子就是给够安全感 -- 段永平
- 同理，自己在有安全感的时候会更有创造力

- 正常流程：分清是谁抛出话题 > 表示关注 > 如果感兴趣则追问细节 > 互相分享 > 兴趣减弱会话自然结束
- 若不是该流程的可能：双方可能 存在某种较为强烈的情绪，存在某种坏习惯

```bash
### 沟通的本质

沟通的目的有两层：**信息传递 + 情感连接**。

### 对话的正常节奏

你抛出话题 → 对方接住（表示关注） → 追问细节 → 你回答细节 → 对方分享相似经历或观点 → 循环

以上循环表示对方对你的话题感兴趣。如果对方接住后不展开，则对话自然结束。

同理，对方抛出话题时，流程也是一样的——关键在于**双向流动**。

### 根据兴趣决定回应方式

| 你的状态      | 回应策略    | 具体做法                                         |
|-----------|---------|----------------------------------------------|
| 想继续聊      | 积极推动节奏  | 提问 + 关联分享，让话题延伸                              |
| 兴趣减弱但仍愿互动 | 切换到共鸣模式 | 简短过渡（如"我明白那种感觉…"），分享相关经历或感受                  |
| 不想继续聊     | 温和收尾    | 简短总结性回应（如"原来如此""听你这么说还挺不容易的"）+ 微笑/点头，让话题自然冷却 |

### 三种对话节奏对比

**正常节奏（螺旋式上升，双向流动）：**
> A 说 → B 问/接 → A 答 → B 关联分享 → A 再问/接 → ……

**独角戏节奏（单向输出，话题中断）：**
> A 说 → B："哦。"（或直接转移到自己身上长篇大论）→ A 失去兴趣

**审讯节奏（缺乏分享，像面试）：**
> A 说 → B 不停提问 → A 不停回答 → B 继续提问 → ……

### Stop Doing List

- 对方接过你的话题但全权引导到自己身上，且不把话题抛回来——说明他对你的话题不感兴趣，可以不聊了
- 明白自己的最初目的是**分享**，而非强行让对方感兴趣
- 对话中注意观察自己的情绪，再观察对方的情绪
```

## 经济

- GDP 增长从 8% 降到 5%，可能意味着近 30% 的人面临就业困难——**增速的微小下降，对应的是大量个体的困境**
- 思考经济学问题时，要避免非黑即白、非此即彼的是非观。取而代之的方法论是**权衡利弊**。非黑即白是一种危险的思维方式，因为它忽视了现实中的折中与代价

## 文化

- 有个说法：**西方社会是平面社会，东方社会是等级社会**
    - 西方：每个人追求超越自己，激发自我潜能——冒险家冒着死亡风险突破极限，动力来自内在
    - 东方：你追求的是"上等级"，本质上是追求比别人强——但当目的是比别人强时，就累了，因为人太复杂、比较永无止境

- 追求"人上人"的困境在于：标准是外部的、相对的、永远在变的。而追求"超越自己"的标准是内部的、绝对的、由自己定义的。前者是零和博弈，后者是正和博弈。

## 科学

- **科学不是立场，不是政治，不为权力妥协**——它是客观事实和规律
- 遵循科学，就是要用**数学的推理逻辑和数据量化**，来定性定量地描述问题、描述事实、解决问题
- 科学的核心精神：**可证伪性**——一个理论如果无法被证伪，就不属于科学范畴
