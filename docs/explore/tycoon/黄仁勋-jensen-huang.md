# 黄仁勋（Jensen Huang）

## 一、个人背景

- **中文名**：黄仁勋
- **英文名**：Jensen Huang（原名 Jen-Hsun Huang）
- **出生日期**：1963年2月17日
- **出生地**：中国台湾省台南市
- **祖籍**：浙江省丽水市青田县
- **国籍**：美籍华人
- **身份**：NVIDIA（英伟达）创始人兼CEO、美国工程院院士

**成长经历**：黄仁勋幼年在台湾成长，约9岁时随家人移民美国。童年时期曾在肯塔基州和俄勒冈州生活，青少年时期曾在餐厅做洗碗工、清洁工等。他从小就展现出对技术和电子产品的浓厚兴趣，曾以购买Apple II电脑为动力坚持打工。

**教育背景**：
- 俄勒冈州立大学电气工程学士（1983年毕业）
- 斯坦福大学电气工程硕士（1990年）
- 获台湾大学、交通大学名誉博士学位

**家庭**：妻子为 Lori Mills，两人育有一子一女。

## 二、职业经历

### 早期职业生涯

- **AMD（超微半导体）**：毕业后加入AMD担任微处理器设计工程师，从事芯片设计工作
- **LSI Logic**：后加入LSI Logic，从事CMOS芯片设计，逐步晋升至核心管理层，积累了芯片行业的深厚经验

### 创立NVIDIA（1993年至今）

- **1993年**：30岁时辞去LSI Logic高薪职位，与Chris Malachowsky、Curtis Priem共同创立NVIDIA，启动资金仅4万美元银行贷款。公司最初命名过程颇具戏剧性，从"NV"（Next Version）中取名，最终定为NVIDIA
- **1995年**：推出首款产品NV1，但市场反响不佳，公司几乎濒临破产
- **1997年**：推出RIVA 128，大获成功，公司扭亏为盈
- **1999年**：发明GPU（图形处理器）概念，重新定义了图形计算行业，推出GeForce 256
- **2006年**：推出CUDA（统一计算设备架构），这是黄仁勋最关键的远见之一——将GPU从图形计算扩展到通用并行计算，当时几乎无人理解这一战略，公司投入约6亿美元
- **2012年起**：CUDA生态逐渐成熟，AI研究者发现GPU特别适合深度学习计算，NVIDIA由此成为AI浪潮的最大受益者
- **2016年**：推出Pascal架构，AI训练需求爆发
- **2020年**：推出Ampere架构，市值突破万亿美元
- **2024年**：推出Blackwell架构，NVIDIA成为全球市值最高的公司
- **2026年3月**：在GTC 2026发布Vera Rubin平台、Groq 3 LPU、Feynman芯片架构等产品

## 三、核心观点与管理哲学

### 关于AI与计算

- "我们正在从'以人写规则的计算'进入'以AI推理为核心的计算时代'"
- "AI不再是单一的突破或应用，而是必不可少的基础设施"
- "Token已成为AI时代的核心商品，推理与执行场景将带来Token消耗的指数级增长"
- "数据中心不再是文件仓库，而是生产Token的'工厂'"

### 关于创业与坚持

- "如果早知道会这么难，我绝不会创办英伟达"
- "你是否相信这件事值得做？你是否愿意从第一性原理开始推演？你是否能在痛苦中坚持学习？你是否有勇气去承受失败的代价？"
- "作为一家公司，作为一个企业家，你必须非常渴望成功，程度更甚于那些渴望你倒下的对手"
- "坦诚面对错误，谦卑寻求帮助，是聪明、成功人士最难学会的"

### 关于管理与文化

- **极致扁平化**：黄仁勋直接管理约40人，几乎没有中间管理层，信息传递迅速，员工被充分赋权
- **使命驱动**："使命才是老板"——不是CEO发号施令，而是公司的使命驱动所有人
- **危机感文化**：核心理念是以赢为目标，积极竞争，避免自满，始终保持危机感
- **透明沟通**：公司内部高度透明，鼓励直接反馈

### 关于人才与技术

- "你此刻在做的事，可能是你人生的重大突破"
- "你们所处的年代很复杂，却也是你们的机会"
- 他始终坚持"做难而正确的事"，顶住巨大压力投入CUDA等长期项目

## 四、最新动态与观点（2025-2026）

### CES 2026（2026年1月）

- **AI革命进入新纪元**：黄仁勋在CES 2026发表主题演讲，提出AI正在从以人写规则的计算转向以AI推理为核心的计算时代
- **物理AI**：推出物理AI技术，通过合成数据训练AI理解物理世界，将AI从数字世界延伸到物理世界
- **自动驾驶**：预言未来5-10年自动驾驶将成为汽车标配
- **芯片架构重构**：AI将重构芯片架构，推动算力革命

### GTC 2026（2026年3月）— 最重磅

- **AI工业化时代到来**：AI产业重心已从"大模型训练"全面转向"推理落地、智能体研发、物理世界交互"
- **五层架构**：形成"能源→芯片→基础设施→模型→应用"的完整产业架构
- **"Token之王"**：英伟达核心目标是追求全球最低的每Token成本，黄仁勋称"英伟达的Token成本在世界上是不可触碰的"
- **万亿营收目标**：预计截至2027年，公司AI芯片营收将达到至少1万亿美元
- **Vera Rubin平台**：由七款芯片组成的最先进AI计算平台，已全面投产
- **Groq 3 LPU**：专为极致低延迟推理设计，首Token延迟小于0.1秒，推理性能为H100的10倍
- **Feynman芯片**：面向2028年的下一代旗舰，台积电1.6nm制程，推理性能为Blackwell的5倍，Token成本降至Blackwell的1/10
- **硅光子革命**：2026年定为硅光子技术商用元年，推出全球首个CPO光电共封装交换机

### 卡内基梅隆大学毕业演讲（2026年5月）

- **对AI与就业的反直觉观点**：告诉毕业生不必担心AI抢走工作，AI会增强而非替代人类能力
- **终身学习**：强调在AI时代，持续学习和适应变化的能力比任何专业技能都重要

### 全球大会（2026年5月）

- **AI已进入良性循环**：AI的发展已形成"计算→模型→应用→需求→计算"的正向循环
- **全面AI化**：NVIDIA自身100%使用Claude Code、Codex和Cursor等AI工具
