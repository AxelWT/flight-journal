# 《被讨厌的勇气》

**岸见一郎 / 古贺史健**

## 一、作者背景

- **时代与国籍**：日本当代哲学家与作家
- **身份**：岸见一郎是阿德勒心理学研究专家，致力于哲学与心理学的普及工作；古贺史健是资深畅销书作家
- **代表作品**：岸见一郎著有《阿德勒心理学入门》《重新认识阿德勒》等；此书是两人的代表作，风靡全球

## 二、核心观点

本书以对话体形式，阐述了阿德勒心理学的核心理念：

1. **目的论**：人的行为不是由过去的原因决定，而是由现在的"目的"驱动
2. **课题分离**：分清自己与他人的课题，不干涉他人的课题，也不让他人干涉自己的课题
3. **共同体感觉**：人的一切烦恼都来自人际关系，一切幸福也来自人际关系
4. **活在当下**：不需要纠结过去，也不必担忧未来，专注于此时此刻

## 三、为什么值得读

- **适合人群**：在人际关系中感到焦虑、过度在意他人评价、难以拒绝别人、总是活在他人期待中的你
- **能解决的问题**：
  - 摆脱"原生家庭决定论"的束缚
  - 学会课题分离，不再为别人的情绪负责
  - 获得"被讨厌的勇气"，活出自我
- **独特价值**：不同于弗洛伊德的"因果论"，阿德勒心理学强调人可以主动选择、主动改变，给了读者改变现状的力量感

## 四、一句精华摘录

> "所谓的自由，就是被别人讨厌。"

💡 **小提示**：这本书以青年与哲人的深夜对话展开，文风轻松易读，却蕴含深刻的人生智慧。
