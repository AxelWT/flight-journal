# 《信息简史》——詹姆斯·格雷克

## 基本信息

- **英文原名**：*The Information: A History, a Theory, a Flood*
- **作者**：詹姆斯·格雷克（James Gleick）
- **英文出版**：2011 年 3 月（Pantheon / Vintage Books）
- **中文出版**：2013 年 11 月，人民邮电出版社（图灵新知系列）
- **获奖**：第九届文津图书奖；入选 2014 年国家新闻出版广电总局"向全国青少年推荐百种优秀图书"

---

## 作者简介

**詹姆斯·格雷克**（James Gleick），1954 年出生于美国纽约，本科毕业于哈佛学院，是当代最具影响力的科普作家之一。

- **新闻生涯**：曾长期在《纽约时报》担任记者和编辑，并多年为《时代》周刊撰写技术专栏。
- **成名作**：1987 年出版首部作品《混沌：开创新科学》（*Chaos: Making a New Science*），入围美国国家图书奖和普利策奖决赛，使"混沌理论"和"蝴蝶效应"的说法家喻户晓，销量超百万册。
- **其他著作**：《费曼传》（1992）、《越来越快》（1999）、《牛顿传》（2003）、《时间旅行》（2016）等，被翻译成二十多种语言。
- **创业经历**：1993 年与合伙人推出 Pipeline 互联网接入服务，率先提供图形界面，是最早一批投入 Internet 创业热潮的人。十四个月后以一千万美元股票出售。
- **写作风格**：以渊博的知识、流畅的叙事、跨学科的视野著称，善于将艰深的科学理论以通俗生动的方式呈现给大众。

## 全书结构

全书共 15 章 + 引言 + 尾声，以"信息"为线索，从远古到当代，横跨语言学、数学、物理、生物、计算机科学等领域。

```
引言 ──► 第1-3章  语言的编码（鼓语 / 文字 / 词典）
       │
       ├──► 第4-6章  计算的萌芽（巴贝奇 / 电报 / 布尔代数）
       │
       ├──► 第7-9章  信息论的诞生与扩展（香农 / 控制论 / 熵）
       │
       ├──► 第10-13章 信息的宇宙化（DNA / 模因 / 随机性 / 量子）
       │
       └──► 第14-15章 + 尾声  洪流与意义的回归
```

## 核心观点

### 1. 信息是文明的核心驱动力

信息不是现代才出现的概念。从非洲鼓语到 DNA 编码，信息始终是人类文明（乃至生命本身）运作的基础。格雷克说："信息是我们的世界赖以运行的东西——血液和燃料，生命的原则。"

### 2. 香农的信息论是分水岭

1948 年香农将"信息"精确量化（比特），并将其与意义脱钩，是思想史上的一次革命。此后，"信息"成为连接计算机科学、生物学、物理学、语言学等学科的通用货币。

### 3. "万物源自比特"（It from Bit）

从麦克斯韦妖到量子信息论，科学越来越表明信息与物理实在密不可分。信息不是抽象概念，而是宇宙的基本构成要素。部分科学家认为，信息比物质和能量更根本。

### 4. 生命本质上是信息处理

DNA 是编码系统，遗传是信息传递，进化是算法过程。薛定谔"以负熵为食"的洞见、沃森和克里克的双螺旋发现，都指向同一个结论：**生命的本质就是信息的存储、复制和处理**。

### 5. 冗余是通讯的必要代价

从非洲鼓语的冗长诗意表达，到英语本身的冗余结构，再到 DNA 中的重复序列——冗余不是浪费，而是对抗噪声、确保信息准确传递的必要机制。这是香农信息论的核心洞见之一。

### 6. 信息过载是永恒的人类困境

信息爆炸不是互联网时代的新问题。印刷术、电报、电话的发明都曾引发信息焦虑。人类一直在发明应对手段——从词典、百科全书到搜索引擎和维基百科。关键不是逃避信息洪流，而是学会在其中导航。

### 7. 历史是"信息逐渐意识到自身"的过程

格雷克写道："每一种新媒介都改变了人类思维的本质……历史就是信息逐渐觉醒的故事。"从文字到电报到互联网，人类在创造信息技术的同时，也在逐步理解信息本身是什么。

## 为什么值得读

> 如果你只能读一本关于"信息"的书，这本就是了。它不只是科技史，更是一部以"信息"为主角的文明史——读完之后，你看待文字、基因、互联网乃至整个世界的方式都会不同。

**它适合这三类人：**

- **对"信息"本质感兴趣的人**——从鼓语到量子比特，一次全景式的思想之旅
- **计算机/通信/生物领域的学习者与从业者**——理解学科的源头与思想脉络
- **喜欢跨学科宏大叙事的读者**——堪与《自私的基因》《枪炮、病菌与钢铁》并列的科普佳作
