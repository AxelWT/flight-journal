## 好照片的三个原则

一幅好照片要具备以下三个要素：

1. **鲜明的主题/主体** — 表现一个人、一件事物，或一个故事情节。主题必须明确，使任何观赏者一眼就能看得出来。
2. **注意力引向被摄主体** — 焦点清晰，主体必须引人注目。
3. **画面简洁** — 只包括有利于把视线引向被摄主体的内容，排除或压缩可能分散注意力的内容。

> 总结：拍照前先思考——主题、焦点、是否够简洁。

## 如何突出主体（第八课）

如何把观众注意力吸引到被摄主体上：

### 1. 布局突出重点

**三分线构图法**：给人以愉快的平衡感和无拘束的宽松感，不像正中央那么死板。除非有充分理由让主体靠边站或位于正中心，否则最好将主体放在中心偏左或偏右的位置。

### 2. 大小比例突出重点

让被摄主体的尺寸看起来比周围物体更大。初学者常见问题是合照时站得太远，导致被摄主体湮没在其他景物中。应靠近拍摄，使被摄主体在画幅中处于支配地位。

### 3. 搭画框突出主体

对于户外远处的大型物体（山峰、建筑物等），可用搭框技法改善构图。常用方法：把近处的树枝摄入画面，赋予画面空间感和距离感。

> 当被摄物体尺寸较小（或距离较远）无法支配画面时，搭画框是吸引注意力的有效技法之一，常用于旅游摄影和建筑摄影。

### 4. 用光突出主体

无论何时，先看光线。观察光线的效果和体验光线的性质。用光突出重点是最有力的手段之一。

### 5. 汇聚线条突出重点

利用环境中的线条引导观众视线指向被摄主体。

### 6. 选择焦点突出重点

## 常见错误及预防

- **取景太近** — 除非另有打算，否则不要让拍摄主体铺满整个画面，四周要留有余地
- **背景杂乱** — 永远仔细观察被摄主体身后的背景
- **镜头畸变** — 广角拉伸显脸大脸尖腿长，长焦压缩显脸平脸宽腿短

## 专业术语速查

**摄影基础**：光圈、快门、ISO、长焦/中焦/短焦、视频升格（慢动作）、视频降格（延时摄影）、RAW 文件、JPG 文件

**后期处理**：曝光、色调、明暗、清晰度、去模糊、对比度、色轮、对比色、互补色、色彩合成、HSL、颗粒、色彩科学、LUT

> 以上术语可自行查阅百科学习。

## 光

> 摄影是用光的艺术。

### 光的三大特征

| 特征 | 说明 |
|------|------|
| **明暗度** | 光的强弱，随光源能量和距离而变化 |
| **方向** | 单一光源时方向好确定，多光源时较复杂 |
| **色彩** | 清晨和傍晚的光色彩更柔和舒适 |

### 光的四个基本方向

| 方向 | 特点 |
|------|------|
| **正面光（平光）** | 拍摄脸部缺乏层次感 |
| **45° 侧光** | 被认为是人像摄影最佳光线，能产生良好立体效果；室内人像主光源常用此角度 |
| **90° 侧光** | 强调光明与黑暗对比的戏剧性光线，又称"结构光线" |
| **逆光** | 易产生艺术效果，高位光源可在被摄者顶部勾勒出明亮轮廓，称为"轮廓光" |

### 现场光

现场光摄影只使用场景中已有的光线（非户外日光），如家用灯光、霓虹灯光、透光的窗户等。

**优点**：富有真实感，被摄对象容易放松。
