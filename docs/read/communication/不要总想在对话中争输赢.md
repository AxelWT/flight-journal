# 不要总想在对话中争输赢

生活中总有这样一群人：朋友之间闲聊要争对错，同事之间讨论要争高低，哪怕是鸡毛蒜皮的小事，也非要辩出个输赢。

## 一、争的是输赢，丢的是共情

总爱争输赢的人，第一个明显的问题，就是缺乏共情能力。他们陷入了**以自我为中心的闭环**。

和别人争论时，他们的注意力从来不放在事情本身，而放在"我必须赢"这个结果上，忽略对方的感受、立场和需求。

- 朋友吐槽工作不顺，本意是寻求安慰，他们却非要争一句"明明是你自己没做好"
- 家人分享生活琐事，他们却非要纠结"你说的不对，应该这样"

共情能力的缺失，让他们看不到对方的情绪需求，只在乎自己的输赢。久而久之，只会被身边的人慢慢疏远。

> 共情的核心，是看见他人。

真正厉害的人，从来不会在无关紧要的小事上争输赢。因为他们懂得，比起争赢对方，照顾他人的情绪、维护彼此的关系，才是更重要的事。

而爱争输赢的人，恰恰不懂这些。他们把共情抛在脑后，用输赢筑起了与他人之间的高墙。

## 二、执着于对错，输的是格局

如果说缺乏共情是表层短板，那么**格局狭隘、过度在意面子**，就是爱争输赢背后的深层硬伤。

那些总爱争输赢的人，是把输赢和面子绑定在了一起。他们怕输，怕的是丢了自己的体面，怕被别人看不起。

- 职场上，为了一句观点分歧，非要跟同事争得面红耳赤，哪怕事后证明自己错了，也不肯低头认错
- 生活中，为了一点鸡毛蒜皮的小事，跟家人争输赢，宁愿冷战几天，也不愿主动退让一步

他们以为这样赢了面子，其实这种斤斤计较的执念，只会暴露自己的格局狭隘。

> 真正有格局的人，不会在小事上纠结对错，更不会为了面子争得你死我活。他们懂得退让、懂得包容。

## 三、陷的是执念，缺的是底气

深究之下，总爱跟人争输赢的本质，其实是**内心的不自信**——无法接纳自己可能不完美、可能会输的真相。

那些非要争赢的人，往往需要通过赢来证明自己的价值，需要通过碾压别人来获得内心的安全感。

他们的内心，藏着一份深深的焦虑：怕自己不够好，怕自己被否定，所以只能通过争输赢来掩饰内心的不安与自卑。

> 接纳自己，接纳自己的不完美，接纳自己可能会输。不必通过争输赢来证明自己，也不必通过碾压别人来获得安全感。

当一个人能够坦然面对输赢，懂得赢不骄傲、输不气馁，才能真正获得内心的自在，也才能拥有更成熟的情商。

## 最后

人生从来都不是一场非赢即输的博弈。很多事情没有绝对的对错，只有不同的立场。

放下争输赢的执念，学会共情、放大格局、接纳自己。你会发现，比起争赢对方，活得自在通透、留住真心，才是人生最大的收获。
