# 你越是放松，结果往往越好

> 你越用力抓结果，结果离你越远。

## 这不是鸡汤，是注意力法则

当你过度在意结果时，皮质醇飙升，注意力收窄到只剩"那个目标"——你盯死了靶心，却忘了怎么拉弓。

你以为自己在努力，其实只是在跟自己较劲。

放弃对结果的执念，不是摆烂。是把注意力从"结果会不会来"挪到"眼前这一步怎么做"。

就像射箭，老手不看靶心，他专注拉弓、呼吸、撒放。动作对了，箭自然中靶。

## 别追逐目标，要建立系统

追逐目标是**用力**，建立系统是**放松**。

系统不依赖意志力——你只管今天要做的那一件事，做完了就结束，明天继续。

找到那件"做了就不想停"的事。难度刚好，能进入心流，产生的价值远大于投入的时间。

那是你的高杠杆活动。

就像滚雪球，找对长坡，轻轻一推，它自己会越滚越大。

## 你的注意力是唯一稀缺资源

别把它浪费在"万一失败了怎么办"上——那些事根本不存在。

你每焦虑一次，能量就漏掉一点。还没开始做，你已经累了。

**三个心法：**

1. 结果与你无关，你只能控制动作
2. 专注当下这一个动作
3. 焦虑来了就让它来，别招待它

用手抓沙子，抓得越紧，流失越快。高手弹钢琴，手是放松的——只有放松，才能弹出最快的音阶。

> 真正的自由，不是得到所有结果，而是不再被结果绑架。

你越是放松，结果往往越好。
