# 《炒股的智慧》— 陈江挺

> 读书笔记

---

## 作者简介

**陈江挺**，华裔投资人，出生于中国，后移居美国。他在华尔街浸润多年，亲历华尔街的牛熊更替，对股票市场的本质有着深刻而朴素的洞察。

他的写作风格极为独特——没有晦涩的理论和公式，而是用大白话将股市的底层逻辑讲透。这本书最初是他在网上写给朋友的投资心得，因为反响热烈才结集出版。他自称不是"大师"，只是一个在市场中活下来的"老手"，这种坦诚本身也是书中智慧的体现。

《炒股的智慧》被誉为**华人炒股入门第一书**，与《股票作手回忆录》并称 trader 必读。

---

## 股票的本质

### 股票是什么

> 股票是人为规定的游戏，既然是游戏就有规则。你只有在规则之内才能赢。

- 股票不是科学，没有精确公式可以预测——它是**概率游戏**
- 股价的短期波动由供需决定，而供需的背后是人性的贪婪与恐惧
- 股票市场是**少数人赚多数人钱的地方**——如果大多数人都对，谁亏钱？

### 股票为什么难

- 股票是**反人性**的——你觉得舒服的操作往往是错的
- 股票是**最自由的职业**，但自由带来的恰恰是最难克服的自律问题
- 市场的信息永远不完整——你永远在不确定性中决策
- 大部分人失败不是因为没有方法，而是因为**无法执行自己的方法**

## 交易哲学

### 炒股的三条基本原则

| 原则 | 含义 |
|------|------|
| **止损** | 亏损超过预设比例，无条件卖出——这是生存底线 |
| **保本** | 活着比赚钱重要，先不亏钱再说赚钱 |
| **一致性** | 用同一套逻辑买卖，不能这次看技术面下次看基本面 |

> 炒股的第一条原则是不要亏钱，第二条是记住第一条。——巴菲特

### 顺势而为

- 趋势是你的朋友——不要和趋势作对
- 上涨趋势中做多，下跌趋势中观望或做空，横盘趋势中休息
- 趋势没变就不要动——**过早行动和过晚行动一样危险**
- 判断趋势的方法不需要复杂：均线、高低点突破就够用了

### 何时买卖

**买入条件（三个同时满足）：**

1. 大盘处于上涨趋势
2. 个股处于上涨趋势
3. 有买入的技术信号（如突破、回踩支撑等）

**卖出条件（满足任一即卖）：**

1. 止损触发——无条件卖出
2. 趋势改变——高位放量滞涨、跌破均线
3. 达到目标价——落袋为安

> 买对卖错只是少赚，买错卖对只是小亏，买错卖错才是大祸。

## 资金管理

这是全书最重要的部分之一，也是大多数散户完全忽视的部分。

### 仓位管理

- **永远不要满仓**——满仓意味着你没有任何容错空间
- 每次交易的仓位不超过总资金的 **20%-25%**
- 同时持有的股票不超过 **5 只**——多了看不过来
- 亏损时减仓，盈利时加仓——**顺势加仓，逆势减仓**

### 止损

> 止损是炒股最重要的技能，没有之一。

- 每笔交易的止损比例不超过 **7%-8%**（总资金的 2% 以内）
- 止损不是止损线被触发才卖——**当买入理由不再成立，就该卖**
- 止损不是认输，而是**为下一次正确操作保留弹药**
- 不止损的后果：小亏变大亏，大亏变深套，深套变绝望

### 盈亏比

- 每次交易的预期盈利至少是止损的 **3 倍**（盈亏比 3:1）
- 10 笔交易中对 3 笔错 7 笔，只要盈亏比够大依然可以赚钱
- **关键是亏的时候亏得少，赚的时候赚得多**
- 很多人的问题不是胜率低，而是赚小钱亏大钱——盈亏比倒挂

### 复利的力量

- 年收益 20%，10 年后资金翻 6 倍
- 年收益 30%，10 年后资金翻 14 倍
- 追求稳定的复利，而非一夜暴富
- **亏 50% 需要赚 100% 才能回本**——这就是大亏损的致命性

## 技术分析

### 技术分析的核心理念

- 技术分析不是预测未来，而是**评估当前趋势的胜率**
- 价格反映一切——所有信息已经包含在股价中
- 历史会重演——因为人性不变

### 关键技术概念

| 概念 | 要点 |
|------|------|
| **支撑与阻力** | 支撑是买方聚集的价格区，阻力是卖方聚集的价格区；突破阻力往往意味着新趋势 |
| **趋势线** | 上涨趋势中连接低点，下跌趋势中连接高点；趋势线被突破意味着趋势转变 |
| **成交量** | 量价配合是健康趋势的标志——价涨量增、价跌量缩；放量滞涨是危险信号 |
| **均线** | 短期均线（5/10/20）看方向，长期均线（60/120/250）看趋势；金叉买入、死叉卖出 |
| **突破** | 价格突破盘整区间或前期高点，伴随放量是最可靠的信号之一 |

### 买入形态

- **突破买入**：价格放量突破前期高点或盘整区间
- **回踩买入**：突破后回踩支撑位不破，确认后介入
- **底部反转**：W 底、头肩底等形态确认后买入

### 卖出形态

- **高位放量滞涨**：成交量放大但价格不涨，主力可能在出货
- **跌破支撑**：关键支撑位被跌破，趋势可能反转
- **急拉之后**：短期内大幅拉升后，往往是阶段性顶部

## 交易心理

这是陈江挺花费最多笔墨的部分——他认为**炒股 80% 是心理，20% 是技术**。

### 人性的敌人

| 心理陷阱 | 表现 | 后果 |
|---------|------|------|
| **贪婪** | 赚了还想赚更多，该卖不卖 | 利润回吐，甚至亏损 |
| **恐惧** | 亏了不敢止损，怕卖了就涨 | 小亏变大亏 |
| **希望** | 亏损时"希望"它会涨回来 | 深套 |
| **懊悔** | 卖了之后涨了就后悔 | 追高买回，又被套 |
| **冲动** | 看到涨就追，看到跌就怕 | 高买低卖，反复割肉 |
| **过度自信** | 连续盈利后觉得自己是股神 | 放松风控，一次亏光 |

### 贪婪与恐惧的循环

> 市场的顶部是由贪婪构成的，底部是由恐惧砸出来的。

- 牛市顶部：人人贪婪，觉得还会涨——该恐惧时贪婪
- 熊市底部：人人恐惧，觉得还会跌——该贪婪时恐惧
- 你的情绪和市场大众一致时，往往就是你犯错的时候

### 独立思考

- 股评、帖子、群消息——**免费的才是最贵的**
- 别人的方法不一定适合你——资金量、风险承受力、时间精力都不同
- 建立自己的交易系统比学会别人的方法更重要
- 真正的交易者是在孤独中做决策的人

### 等待的智慧

- 一年中真正值得交易的时间可能只有几个月
- 大部分时间应该**空仓等待**
- 不做不是无能，而是在等高胜率的机会
- 耐心等待买入信号，耐心等待卖出信号——**耐心是交易者最大的美德**

## 交易系统

### 为什么要建立交易系统

- 没有系统，你的交易全凭感觉——感觉是最不可靠的东西
- 系统让你在情绪波动时仍然有据可依
- 交易系统不需要完美，只需要**可执行、可重复**

### 交易系统的要素

| 要素 | 内容 |
|------|------|
| 选股规则 | 什么样的股票我会买？（趋势、基本面、板块等） |
| 买入规则 | 什么条件下买入？（突破、回踩、信号确认等） |
| 止损规则 | 什么条件下无条件卖出？（固定比例、技术位等） |
| 卖出规则 | 什么条件下止盈？（目标价、趋势改变等） |
| 仓位规则 | 每次买入多少？最多持几只？ |
| 休息规则 | 连续亏损后暂停交易多久？ |

### 交易日志

- 每笔交易记录：买入理由、卖出理由、盈亏结果、心理状态
- 定期回顾：哪些错误在反复犯？
- 交易的本质是**用小亏试错，用大赚覆盖**
- 不要追求每笔都赚——追求的是总体盈利

## 新手最常见的错误

1. **不止损** — 亏了死扛，最终深套
2. **频繁交易** — 每天都操作，手续费和失误率都很高
3. **追涨杀跌** — 涨了才买（高位接盘），跌了才卖（低位割肉）
4. **满仓操作** — 一把梭哈，毫无回旋余地
5. **听消息炒股** — 你听到的消息已经是第 N 手了
6. **赚钱拿不住，亏钱死不放** — 盈亏比倒挂
7. **没有交易系统** — 随心所欲，全凭感觉
8. **不总结不复盘** — 同样的错误反复犯

## 金句

> 炒股不是赌大小，而是概率游戏。长期来看，胜率站在有纪律的人一边。

> 止损不是认输，而是认错。不肯认错的人，市场会帮你认，代价是十倍百倍。

> 市场上最贵的一句话是"这次不一样"。

> 牛市让人觉得自己是天才，熊市让人发现自己是傻子。

> 交易的秘密不在于你知道什么，而在于你能否做到你知道的。

> 简单的方法执行到位，远胜复杂的方法执行不了。

> 你不是在和股票较劲，你是在和自己较劲。

> 控制风险的人最后赚了钱，追求利润的人最后亏了钱。
