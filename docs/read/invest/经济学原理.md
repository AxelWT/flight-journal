# 经济学入门笔记

> 最重要 / 最基础的经济学概念整理。

---

## 核心概念

| 概念 | 说明 |
|------|------|
| **供给与需求** | 理解几乎所有市场现象的基础，价格由二者的相互作用决定。 |
| **机会成本** | 为了得到某样东西而放弃的其他东西的最高价值，是理性决策的核心。 |
| **边际** | 分析增量和决策的关键（边际成本、边际收益、边际效用），理性决策基于边际量而非总量。 |
| **均衡** | 市场力量达到平衡的状态，是分析市场结果和政策效果的基准点。 |
| **弹性** | 量化一个变量对另一个变量变化反应程度的工具，如需求价格弹性衡量需求量对价格变化的敏感度。 |
| **GDP** | 国内生产总值，一定时期内一国生产的最终物品和服务的市场价值总和，衡量经济规模的核心指标。 |
| **通货膨胀** | 物价总水平的持续上升，直接影响购买力与宏观经济稳定。 |
| **失业率** | 失业人口占劳动力人口的比率，反映资源利用程度和社会福利水平。 |
| **市场失灵** | 市场无法有效配置资源的情况（如污染、公共物品、信息不对称），是政府干预的理论依据。 |
| **比较优势** | 一国生产某种物品的机会成本低于另一国，国际贸易理论的核心基石。 |

## 微观经济学（Microeconomics）

**研究对象**：单个经济主体（家庭、企业）的决策行为。

**核心概念**：
- **需求侧**：效用、边际效用递减、无差异曲线、预算约束线、消费者剩余
- **供给侧**：生产函数、边际产量、边际成本、平均成本、利润最大化、生产者剩余
- **市场**：供给与需求、均衡价格、弹性（需求价格弹性、需求收入弹性、供给弹性）
- **市场结构**：完全竞争、垄断、寡头垄断、垄断竞争
- **成本理论**：机会成本、显性成本、隐性成本
- **市场失灵**：外部性、公共物品、信息不对称

## 宏观经济学（Macroeconomics）

**研究对象**：经济总量及其运行规律。

**核心概念**：
- **总量指标**：国内生产总值（GDP）、国民生产总值（GNP）、潜在产出
- **经济波动**：经济周期（繁荣、衰退、萧条、复苏）
- **就业与物价**：失业率、通货膨胀、通货紧缩、消费者物价指数（CPI）、生产者物价指数（PPI）
- **政策工具**：
  - 财政政策（政府支出、税收、财政赤字、政府债务）
  - 货币政策（利率、存款准备金率、公开市场操作、货币供应量）
- **总供需**：总需求、总供给、菲利普斯曲线
- **开放经济**：汇率、国际收支平衡、外汇储备

## 国际经济学（International Economics）

**研究对象**：国家之间的经济活动与相互依赖关系。

**核心概念**：
- **贸易理论**：比较优势、绝对优势
- **贸易政策**：贸易壁垒（关税、配额）、自由贸易区、关税同盟、共同市场
- **汇率与货币**：汇率制度（固定汇率、浮动汇率）、购买力平价、外汇市场、汇率风险
- **国际收支**：经常账户、资本与金融账户、贸易顺差 / 逆差
- **全球化**：外商直接投资（FDI）

## 金融学（Finance）— 经济学的重要应用分支

**研究对象**：资金融通、资产定价与风险管理。

**核心概念**：
- **时间价值**：货币的时间价值、现值、终值、复利、单利
- **投资与风险**：风险与收益、投资组合、分散化、资产定价（CAPM 模型）、有效市场假说
- **金融工具**：股票、债券、衍生品（期货、期权）
- **利率**：利率期限结构
- **公司金融**：资本结构、股利政策

## 行为经济学（Behavioral Economics）

**研究对象**：心理因素对经济决策的影响，是对传统理性人假设的修正。

**常见偏差**：锚定效应、损失厌恶、从众效应、过度自信、现状偏见、框架效应。

行为经济学的兴起说明：经济学模型虽然建立在理性假设之上，但真实世界中人的决策往往受到认知偏差和社会心理的影响。理解这些偏差有助于做出更好的个人决策和公共政策。

## 经济学十大原理

以下内容出自 N. Gregory Mankiw 的《经济学原理》，是理解现代经济学的起点。

### 人们如何做出决策

**原理一：人们面临权衡取舍**

经济学中，「权衡取舍」是最基本的出发点：要得到一样东西，通常就得放弃另一样东西。经典的例子是「效率 vs. 公平」——效率要求把蛋糕做大，公平要求把蛋糕分匀，两者往往不可兼得。

**原理二：某种东西的成本是为了得到它所放弃的东西**

即机会成本。举例来说，读研究生的成本不只是学费和书本费，更关键的是为此放弃的几年工作收入——那才是真正的机会成本。做决策时，不妨问问自己：我的下一个最佳选择是什么？

**原理三：理性人考虑边际量**

生活中的决策很少是非黑即白的，更多是灰色地带——多一分还是少一分的问题。吃午饭时，你面对的不是「完全不吃」和「大吃一顿」的选择，而是「是否再多吃一口」。边际分析（比较边际收益与边际成本）是理性决策的核心方法。

**原理四：人们会对激励作出反应**

不要低估激励的威力。价格变动、税收调整、补贴政策都会改变人们的行为。政策制定者尤其需要审视激励设计，因为非预期后果（unintended consequences）往往源于被忽视的激励。

### 人们如何相互交易

**原理五：贸易可以使每个人的状况都变好**

贸易不是零和博弈。通过比较优势实现分工合作，可以提升整体效率。举例来说：即便甲在所有工作上的能力都不如乙，乙仍然可以与甲交易——乙专注自己效率最高的事，把相对低效的事交给甲，结果双方都受益。

**原理六：市场通常是组织经济活动的一种好方法**

市场通过「看不见的手」（亚当·斯密，1776）——即价格信号——来引导资源配置。价格差产生套利机会，而套利行为又消除价格差，推动市场趋向动态平衡。

**原理七：政府有时可以改善市场结果**

当市场失灵时（如污染等外部性、国防等公共物品、二手车市场等信息不对称），政府干预有潜力改善结果。但政府本身同样可能失灵，需要注意小政府模式与大政府模式之间的权衡。

### 整体经济如何运行

**原理八：一国的生活水平取决于它生产物品和服务的能力**

生活水平的根本决定因素是生产率——即每单位劳动投入所能产出的物品和服务数量。从国际贸易看，各国通过进出口发挥比较优势，提升整体生产率。从长期看，经济增长的核心驱动力正是劳动生产率的持续提升。

**原理九：当政府发行过多货币时，物价上升**

物价全面持续上涨即通货膨胀。消费者物价指数（CPI）是衡量通胀的常用指标。区分名义利率与实际利率至关重要：实际利率 ≈ 名义利率 − 通货膨胀率。当货币供给增速远超经济产出的增长速度时，「过多的货币追逐过少的商品」，物价必然上涨。

**原理十：社会面临通货膨胀与失业之间的短期权衡取舍**

短期中，通胀与失业呈反向关系，即菲利普斯曲线。政策制定者面临取舍：扩张政策可降低失业但推高通胀，紧缩政策可抑制通胀但推高失业。但长期中这一关系不成立——长期失业率由自然失业率决定，不受通胀影响。

## 重要经济学流派

| 流派 | 代表人物 | 核心主张 |
|------|----------|----------|
| **古典经济学** | 亚当·斯密、大卫·李嘉图 | 市场自行调节，政府不干预，「看不见的手」 |
| **凯恩斯主义** | 约翰·梅纳德·凯恩斯 | 经济可能陷入非充分就业均衡，需要政府通过财政/货币政策干预 |
| **货币主义** | 米尔顿·弗里德曼 | 货币供应量是影响经济的主要因素，主张稳定的货币增长规则 |
| **新古典综合派** | 保罗·萨缪尔森 | 将凯恩斯宏观分析与新古典微观分析结合，是现代主流经济学的基石 |
| **奥地利学派** | 弗里德里希·哈耶克、路德维希·冯·米塞斯 | 强调市场过程和个人主观价值，反对政府干预 |
| **制度经济学** | 道格拉斯·诺斯、罗纳德·科斯 | 制度（法律、产权、习俗）对经济绩效起决定性作用 |

## 参考资料

### 经典教材

- **Mankiw, N. G.《经济学原理》（Principles of Economics）**：十大原理的直接来源，全球最广泛使用的经济学入门教材。推荐原版或梁小民译本。
- **Samuelson, P. A. & Nordhaus, W. D.《经济学》（Economics）**：首部将宏观与微观融合的经典教材，体系全面。
- **曼昆《宏观经济学》**：学习宏观经济学的标准教材。
- **范里安《微观经济学：现代观点》**：中级微观经济学的标准教材，数学要求适中，逻辑严谨。

### 通识读物

- **《经济学通识》—— 薛兆丰**：用通俗语言讲解经济学思维方式，适合建立直觉。
- **《魔鬼经济学》（Freakonomics）—— Levitt & Dubner**：用经济学视角分析日常生活中的有趣问题。
- **《国富论》—— 亚当·斯密**：现代经济学的奠基之作，「看不见的手」的原始出处。
- **《思考，快与慢》—— Daniel Kahneman**：行为经济学经典读物，深入探讨认知偏差。

### 扩展阅读

- **《经济学规则》—— Dani Rodrik**：讨论经济学的适用范围与局限。
- **《贫穷的本质》—— Banerjee & Duflo**：2019 年诺贝尔经济学奖得主作品，用实验方法研究发展经济学。
- **《货币崛起》—— Niall Ferguson**：以金融史视角讲解货币与信用的演变。
- **《金融炼金术》—— George Soros**：反思有效市场假说，提出反身性理论。

### 在线资源

- **Coursera / edX**：MIT、Harvard 等名校的经济学开放课程。
- **Khan Academy 经济学**：免费中英文视频课程，概念讲解清晰直观。
- **Investopedia**：英文经济 / 金融百科，适合快速查阅术语。
- **知乎 / 得到 App**：中文经济学讨论与专栏内容。

### 学术资源

- **NBER（National Bureau of Economic Research）**：美国顶尖经济学研究机构，发布大量工作论文。
- **SSRN（Social Science Research Network）**：社会科学预印本平台，经济学论文更新及时。
- **JSTOR / Google Scholar**：学术论文检索工具。
