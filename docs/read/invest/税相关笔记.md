# 税相关笔记

## 直接税和间接税

### 直接税与间接税对比

| 类型 | 税种 | 说明 |
|------|------|------|
| **直接税** | 个人所得税 | 对个人工资、稿酬等所得征税，不可转嫁 |
| | 企业所得税 | 对企业利润征税，理论上可部分转嫁，但实践中归为直接税 |
| | 财产税类（房产税、契税等） | 对持有或转移的财产征税 |
| **间接税** | 增值税 | 对商品增值环节征税，最终由消费者承担 |
| | 消费税 | 对特定商品（如烟草、酒类）征收，含于售价中转嫁消费者 |
| | 关税 | 进出口环节征收，影响商品最终价格 |

| 方面 | 直接税 | 间接税 |
|------|--------|--------|
| **功能侧重** | 调节收入分配，促进社会公平（如累进税率） | 筹集财政收入效率高，对经济干预灵活 |
| **征管难度** | 需精准核算所得/财产，征管复杂（如个税需综合申报） | 仅监管流转额，征管简便（如增值税凭发票抵扣） |
| **纳税人感知** | 税负透明，痛感明显（从收入中直接扣除） | 税负隐含于价格，消费者不易察觉 |

### 我国税制结构现状

- **以间接税为主**：增值税、消费税等占比高，商品价格中含大量隐性税负。
- **改革方向**：逐步提高直接税比重（目前约 35%），优化个人所得税（如综合与分类结合）、完善房产税、研究遗产税；平衡公平与效率，避免过度影响经济活力。

## 基尼系数

基尼系数是衡量收入或财富分配不平等程度的核心指标。取值在 **0～1 之间**：
- **0**：收入绝对平均（所有人收入完全相同）
- **1**：收入绝对不平均（所有收入归一人所有）

实际社会中，两种极端情况均不存在，数值通常介于 0.2～0.6 之间。

| 基尼系数范围 | 收入分配状态 | 社会意义 |
|-------------|-------------|----------|
| < 0.2 | 绝对平均 | 高度均等化，但可能抑制经济活力 |
| 0.2～0.3 | 比较平均 | 相对平等，社会稳定性高 |
| 0.3～0.4 | 相对合理 | 多数发达国家的理想区间 |
| **0.4～0.5** | **差距较大** | **国际警戒线，需政策干预** |
| > 0.5 | 差距悬殊 | 高风险社会不稳定 |

### 中国的基尼系数现状

- **2022 年**：**0.467**（居民人均可支配收入基尼系数），较 2012 年（0.474）略有下降，但仍高于警戒线。
- **2025 年预测**：约 **0.466**，显示收入差距持续显著。
