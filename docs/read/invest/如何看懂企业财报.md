# 如何看懂企业财报

财务报表分为**资产负债表**、**利润表**和**现金流量表**，俗称三大报表。

> 股票价格理论上等于市场预期未来总股利的折现。

---

## 资产负债表

资产负债表是在某个时间点（一般为会计期末）当天企业的资产、负债以及所有者权益的情况。

**恒等式**：资产 = 负债 + 所有者权益

- **资产**：预期会给企业带来经济利益的资源，例如企业厂房、设备、流动资金等。
- **负债**：企业的债务，分为**流动负债**（一年内需要偿还）和**非流动负债**（偿还期限在一年以上）。
- **所有者权益**：即股东权益。

### 短期偿债能力指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **流动比率** | 流动资产 / 流动负债 | 越大表示短期偿债能力越强；过大则可能未充分利用财务杠杆 |
| **速动比率** |（流动资产 − 存货）/ 流动负债 | 剔除存货影响，更能体现短期偿债能力 |
| **现金比率** |（货币资金 + 有价证券）/ 流动负债 | 剔除存货和应收账款，最能反映直接快速偿付流动负债的能力 |

### 长期偿债能力指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **资产负债率** | 总负债 / 总资产 | 衡量企业负债水平 |
| **利息保障倍数** | 息税前利润 / 利息费用 | 衡量企业支付借款利息的能力 |

## 利润表

利润表体现的是公司利用已有资产挣钱的能力，反映企业在一段时间内经营所得到的回报（动态情况）。

大多数利润表遵循**权责发生制**——如果企业已经做了义务上该做的事，即使没有收到货款也会先记入收入，反之亦然。简单来说，就是只看事情做没做，不看钱到没到位。

**恒等式**：收入 − 成本 = 利润

**收入科目**：
- **营业收入**：可持续，更能反映企业赚钱的能力。
- **营业外收入**：未必可以一直保持，不能与营业收入一概而论。

**成本费用**：营业成本、销售费用、管理费用、财务费用。

**利润项目**：营业利润、利润总额（营业利润 + 营业外利润）、归属母公司股东的净利润、少数股东权益。

### 盈利能力指标

| 指标 | 公式 |
|------|------|
| **毛利率** |（营业收入 − 营业成本）/ 营业收入 |
| **营业利润率** | 毛利率 − 期间费用率（销售、管理、研发费用） |
| **净利润率** | 营业利润率 − 利息/税费 ± 非经常性损益率 |

---

## 现金流量表

利润表中的收入包含了实际未收到的销售款（应收账款），或者减去了打算要付却实际未付的费用（应付账款）等。很多时候利润表中的利润有可能不会变现。

现金流量表遵循**收付实现制**，只记录真实的资金流动。正向现金流表示现金流入企业，反之表示流出。

> 只有利润是不够的，见到真钱才是硬道理。

**现金流来源**：
1. **经营活动产生的现金流**：影响利润变化的主营业务活动。流入：销售产品；流出：采购、发放薪水。
2. **投资活动产生的现金流**：处置长期资产、收回投资、取得投资收益等。流入：取得投资收益收到的现金、现金股利和债券利息等；流出：购买厂房等。
3. **融资活动产生的现金流**：借款、还款和发行新股募集资金等。流入：向银行借款、吸收投资；流出：归还借款本息、支付现金股利等。

---

## 与投资相关的指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **市盈率（PE）** | 股价 / 每股盈利 = 公司市值 / 年度净利润 | 股票的回本年限，越高表示越「贵」 |
| **市净率（PB）** | 股价 / 每股净资产 | 投资者以高于净资产多少倍的价格买入，越高「成本」越高 |
| **净资产收益率（ROE）** | 净收入 / 股东权益 | 衡量企业用本金挣钱的能力 |
| **资产收益率（ROA）** | 净收入 / 总资产 | 衡量公司资产使用效率，常用于资本密集型行业对比 |
