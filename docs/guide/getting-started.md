# 快速开始

欢迎使用 VitePress 博客！

## 创建内容

在 `docs` 目录下创建 `.md` 文件即可添加新页面。VitePress 使用基于文件的路由，目录结构即页面路径。

## 开发

启动开发服务器：

```sh
npm run docs:dev
```

## 构建

构建静态文件：

```sh
npm run docs:build
```

构建产物在 `docs/.vitepress/dist` 目录下。

## 预览

预览构建结果：

```sh
npm run docs:preview
```
